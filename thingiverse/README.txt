# Summary

I've got a pair of 20mm short [Parf Dogs](http://www.leevalley.com/en/wood/page.aspx?p=71185&cat=1,41637), I've got a [Veritas Quad T-Slot Track](http://www.leevalley.com/en/wood/page.aspx?p=73519&cat=1,43455,61994,73519).  Merge.


# Post-Printing

Needs a bit of sanding of the Parf Dog hole.  Just roll up some 220 grit and run it through until the dog can slide in.

For each assembly, you'll need a 1/4-20x1" T-Bolt, washer, and 1" dia or less insert knob.  These you usually can get from the 1/4-20 jig hardware set.
